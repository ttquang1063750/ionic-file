## Install Ionic (4) & cordova
```
npm install -g ionic
npm install -g cordova
```

## Install module
```
  cd ionic-file && npm i
```

## Broswer dev
```
ionic serve --o
```

## Build app
```
ionic cordova build android --prod --release --no-insight --verbose
ionic cordova build ios --prod --release --no-insight --verbose
```
